자바 UI 구현에 쓰이는 대표적인 것을 여러가지 검색해보니
보이는 대표적인 것으론 Swing,AWT,SWT가 있다.

AWT와 SWT 는 운영체계 환경을 따르기 때문에 실행하는 곳에 따라 디자인이 다를수 있다. Swing는 자바환경을 따라서 어디서 실행하든 디자인 이 동일하다.

둘 중 하나를 사용하는 것도 가능하고 둘다 사용하는 것도 가능하다.

하지만 같이 사용할때 주의 할 점은 AWT로 생성한 UI는 SWING UI에 가려진다.

전체적으로 읽어봤을때는 SWING이 나을 것같다. ( 민규생각 )


awt는 java.awt.* 에 존재

swing은 javax.swing.*에 존재
